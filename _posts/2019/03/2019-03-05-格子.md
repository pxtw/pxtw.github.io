<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
原来，我住在这里。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
抬头就是天，低头就是草地。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
打石子，坐在欢欢身边陪他说话是我最快乐的事情。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
最大的烦恼，也无非是暗自发恨地恼怒，关于妈妈总是晚回来，家成了只睡觉的地方。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
<br /></div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
后来，我走了，狗死了。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
我就再没怎么回来过。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
<br /></div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
现在，你还住在这里。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
那个摆夜市的，把狗原来的住处也霸占了。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
你佝偻着，看外面的雨。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
抬头不再是天，而是更多的这里，像一个个密密麻麻的冻住的格子。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
最大的烦恼成了忧愁未来和你还是这个样子。</div>
<div style="background-color: white; color: #222222; font-family: georgia, times, serif; font-size: 12.8px; margin-bottom: 1em; margin-top: 1em;">
打石子，坐在欢欢身边陪他说话是我最快乐的事情。</div>
