1. Github clone repository 

2. 编辑文件

3. 把私钥找到，位置确定好

4. 在`~/.ssh/config`里配置私钥，样板如下

`
Host pxtw
Hostname github.com
IdentityFile /Users/tempuser/id_rsa
IdentitiesOnly yes
`

5. git add, git commit

6. git remote set-url origin git@pxtw:pxtw/pxtw.github.io

7. git push 或者 git push -v origin
